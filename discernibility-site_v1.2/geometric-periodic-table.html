The Geometric Periodic Table

Two interactive 3D views of the 120° structure

The periodic table is not a flat grid—it's a three-dimensional geometric structure where each period occupies 120° of arc, with elements organized radially by d-electron count and group.

Below are two complementary 3D visualizations of the same underlying geometry. Each emphasizes different aspects of the structure.

The Shared Principle: 120° Per Period

Both visualizations are based on the same geometric foundation:

  • 120° per period - Three periods complete a full 360° rotation
  • Radial organization - Position determined by d-electron count (Groups 4-6: d=2,3,4)
  • Angular ranges - s-block (0°-17°), d-block (17°-34°), p-block (34°-60°), noble gases (120°)
  • f-block interleaving - Lanthanides and actinides nested within d-block angular range

Why This Geometry?

The 120° structure emerges from the constitutional coordinate system where elements are positioned by their d-electron count (d), Period (P), and bonding character (g). These coordinates not only organize the periodic table geometrically, but also predict material properties with sub-2% accuracy.

φ = f(d, P, g) where 0° ≤ φ ≤ 120° per period

The same coordinates that position elements in these visualizations are used to predict elastic moduli, melting temperatures, and other properties of 70+ refractory compounds.
